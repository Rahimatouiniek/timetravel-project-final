timetravel-project-final    


Application Web interactive est une vitrine technologique pour une agence de voyage dans le temps. Elle utilise une architecture moderne basée sur l'intégration de ressources externes pour optimiser les performances et la fluidité. Crée par  IA génératives 

Liens Officiels
Lien du site live: Visiter la Webapp (https://timetravel-agency-lu-m4a3.bolt.host)
Lien du code : Dépôt GitHub Public (https://github.com/Rahimatouiniek/timetravel-project-final)
Projet finalisé par Rahimatou Iniek TAPSOBA et Jordan ZOKPODO.


## Stack Technique
Frontend : React.js (v18), TypeScript, Vite.
Stylage : Tailwind CSS pour une interface responsive et moderne.
Architecture : Code organisé en composants réutilisables sous `/src/components`.

Imgur :Hébergement Images, Utilisation de liens directs pour intégrer les visuels des destinations historiques dans le code source.
Cloudinary :Flux Vidéo, Exportation et intégration des vidéos promotionnelles via des URLs optimisées pour le streaming.
Bolt.new :Déploiement Environnement de développement cloud utilisé pour générer l'application et l'héberger en direct.


## Features
- Landing page interactive
- Galerie de 3 destinations temporelles
- Chatbot IA conversationnel
